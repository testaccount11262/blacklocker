WallpaperChanger - a program for change wallpaper on time or manually (click on tray). The program is free and open source software.
Supported features:
- Styles of filling of wallpaper: 
Center, 
Tile, 
Stretch,
Fit (resized while maintaining the aspect ratio),
Fill (resized and cropped while maintaining the aspect ratio)
- Usage of scaling of wallpaper Windows (Windows 7, Vista) or own.
- Background color:
Automatic (prevailing color of edges is defined),
Single-color,
Gradient,
Stretch edges
- Assignment of hot key for change of wallpaper
- Not to change wallpaper at high CPU loading
- Not to change wallpaper at idle time
- Show pop-up balloon notification

Please report bugs or enhancements requests to the sourceforge project page or email dmitryborisov@users.sourceforge.net.

WallpaperChanger - программа для изменения обоев по времени или вручную (щелчком на трее). Программа бесплатная и с открытым исходным кодом.
Поддерживаемые функции:
- Стили заполнения обоев:
По центру,
Замостить,
Растянуть,
По размеру (масштабировать сохраняя соотношение сторон),
Заполнить (масшатабировать и обрезать сохраняя соотношение сторон)
- Использование масштабирования обоев Windows (Windows 7, Vista) или собственный.
- Цвет фона:
Автоматический (определяется преобладающий цвет краев),
Одноцветный,
Градиент,
Растягивание краев
- Назначение горячей клавиши для смены обоев
- Не менять обои при высокой загрузке процессора
- Не менять обои при простое
- Показ всплывающих уведомлений

Если вы найдете какой-нибудь баг, или у вас есть предложения по улучшению, то, пожалуйста, напишите на dmitryborisov@users.sourceforge.net.
